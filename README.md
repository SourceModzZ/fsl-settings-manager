## 📌 Overview

Editing configuration files manually can often lead to syntax errors, broken setups, or frustration. **FSL Settings Configurator** provides a clean graphical user interface (GUI) that allows users to seamlessly modify their `settings.ini` file without touching a single line of raw text.

---

## ✨ Features

* 🛠️ **User-Friendly Interface:** Easily tweak all configuration parameters using toggles, sliders, and input fields.
* 📁 **Auto-Detection:** Automatically locates your existing `settings.ini` file.
* 💾 **Safe Saving:** Formats and writes your preferences cleanly to avoid broken configuration files.
* 🛡️ **Error Prevention:** Built-in validation prevents invalid inputs or syntax mistakes.
